# QQ 聊天记录统计分析与应用

#### 1、关于项目

​	本项目围绕 qq 聊天记录的统计、分析与具体可视化应用展开。目前正在开发中。



#### 2、进展与规划

已完成开发的子项：

  - [基于所有聊天记录生成词云](https://github.com/AiCorein/QQChatHistory_Analysis/tree/main/wordCloud)

未来可能开发的子项：

- 基于所有聊天记录，生成可视化的情感、话题图表
- 基于个人发送消息生成词云、可视化的情感、话题图表
- 基于聊天记录资料构建的自动聊天 bot
